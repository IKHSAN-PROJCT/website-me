di sini kalian cari di google atau di chrome juga bisa ya dan nama website nya github.com

Oke sekarang kalian masuk ke halaman website nya , dan disini kalian harus sign up dulu kalau kalian belum punya sama sekali akun github-nya dan jika kalian sudah mempunyai langsung login saja github.login

Oke tahap selanjutnya kalau kalian sudah berada di halaman dashboard atau home , kalian scroll ke bawah dan buat repository nya

Beri nama repository yang kalian buat oke lalu klik creat a new repository

Disini kalian pilih uploading a an exiting file

Disini file kalian upload ya klik choose your files

Dan setelah itu kalian pilih file html yang akan di upload ke guthub bebas tergantung kalian ya , dan tenang aja github bisa upload berbagai file

setelah di upload kalian klik yang berwarna hijau ya
